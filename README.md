# Fenmo Expense Tracker

## Description
A minimal full-stack expense tracker with FastAPI backend and Streamlit frontend.

## Tech Stack
- Python
- FastAPI
- SQLite
- Streamlit

## Setup

### Backend
cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload  

### Frontend
cd frontend  
pip install -r requirements.txt  
streamlit run app.py
